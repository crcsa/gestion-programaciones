# Diagramas Excalidraw — Sistema de Gestión de Programaciones
## Cruz Roja Colombiana — Banco de Sangre

Cada archivo `.json` contiene un array de elementos Excalidraw editables.
Para usarlos, pega el contenido en [excalidraw.com](https://excalidraw.com) o en cualquier instancia compatible.

---

| # | Archivo | Diagrama |
|---|---------|----------|
| 1 | `diagrama_1_elements.json` | **Arquitectura Modular** — 11 módulos en 5 capas (Seguridad, Operaciones Core, Reglas de Negocio, Integraciones, Soporte) |
| 2 | `diagrama_2_elements.json` | **Flujo de Integración** — CRM → Excel → Plataforma → Hexabank (flujo comercial semanal + integración bidireccional) |
| 3 | `diagrama_3_elements.json` | **Ciclo de Vida de Campañas** — Estados (Tentativa → Confirmada / Cancelada), Tamaños (S/M/L) y 5 Modalidades |
| 4 | `diagrama_4_elements.json` | **Asignación Inteligente de Personal** — Motor de 9 validaciones (3 BLOCK + 6 WARN) con flujo de decisión |
| 5 | `diagrama_5_elements.json` | **Modelo de Datos** — 9 entidades principales en 3 dominios (Identidad, Operaciones, Control de Horas) |
| 6 | `diagrama_6_elements.json` | **Roles y Permisos** — Jerarquía de 4 roles del sistema + Coordinador de Campaña (rol dinámico) |
| 7 | `diagrama_7_elements.json` | **Cómputo Integrado de Horas** — Balance semanal (44h+12h) y límites mensuales (domingos, nocturnos) |
